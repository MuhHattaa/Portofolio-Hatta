Tentang Saya

Halo! Nama saya Muhammad Hatta. Saya menikmati membuat hal-hal yang hidup di internet — mulai dari aplikasi kecil sampai sistem yang bisa diskala. Ketertarikan saya pada pengembangan perangkat lunak dimulai sejak saya bereksperimen dengan project IoT dan aplikasi web sederhana.

Sepanjang waktu, saya berkesempatan bekerja pada berbagai project yang menantang — dari perangkat embedded sampai antarmuka pengguna. Fokus saya saat ini adalah membangun sistem yang andal, dapat diuji, dan mudah dipelihara. Saya juga menyukai proses pembelajaran berkelanjutan dan sering bereksperimen dengan toolchain modern.

Di luar pengkodean, saya suka membaca buku teknis, mengikuti perkembangan teknologi terbaru, dan berkontribusi pada komunitas open source. Saya percaya bahwa pengembangan perangkat lunak adalah tentang kolaborasi dan pembelajaran terus-menerus.
